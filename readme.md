# No-Show Risk Predictor

## Overview

This is a Flask-based prototype that predicts patient no-show risk using a simple weighted scoring model.

## Features

* Input patient characteristics
* Calculates a risk score
* Outputs Low / Medium / High risk
* Shows contributing factors

## Variables Used

* Prior no-shows
* Lead time
* Insurance type
* Reminder confirmation
* Age group

## How to Run

pip install flask
python app.py

Then open: http://127.0.0.1:5000/

## Note

This is an educational prototype and not a clinically validated model.
